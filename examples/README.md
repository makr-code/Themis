# ThemisDB Examples

This directory contains example code and usage demonstrations for ThemisDB.

## Available Examples

Examples demonstrating various ThemisDB features and use cases will be added here.

## Categories

- **Basic Operations** - CRUD operations, simple queries
- **Advanced Queries** - Complex AQL queries, graph traversal, vector search
- **Integration** - Integration examples with other systems
- **Security** - Encryption, authentication, authorization examples
- **Performance** - Optimization techniques and best practices
