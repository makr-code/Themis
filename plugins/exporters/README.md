# Exporter Plugins

Data export plugins for ThemisDB.

## Available Exporters

### JSONL LLM Exporter
**Path:** `jsonl_llm/`

Export data in JSONL format optimized for Large Language Model (LLM) training.

## Features

- Export documents for LLM fine-tuning
- LoRA adapter metadata generation
- vLLM integration support
- Configurable field selection
