# Importer Plugins

Data import plugins for ThemisDB.

## Available Importers

### PostgreSQL Importer
**Path:** `postgres/`

Import data from PostgreSQL databases into ThemisDB.

## Features

- Schema mapping and transformation
- Batch import operations
- Incremental import support
- Data type conversion
