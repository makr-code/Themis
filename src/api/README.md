# API Module

HTTP API server implementation for ThemisDB.

## Components

- HTTP server using Crow/beast
- RESTful API endpoints
- Request/response handling
- API middleware

## Features

- Document CRUD operations
- AQL query execution
- Graph operations
- Authentication and authorization
- TLS/SSL support
