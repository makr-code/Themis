# Index Module

Index implementations for ThemisDB's multi-model database.

## Components

- Adaptive index
- Vector index (HNSW)
- Graph index
- Property graph
- Secondary indexes
- GNN embeddings

## Features

- HNSW-based vector similarity search
- Graph traversal indexing
- Property graph model support
- Adaptive index selection
- Multi-dimensional indexing
