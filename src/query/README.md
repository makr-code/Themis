# Query Module

AQL query parser, optimizer, and execution engine for ThemisDB.

## Components

- AQL parser
- AQL translator
- Query engine
- Query optimizer
- Query parser (generic)
- Semantic cache integration

## Features

- AQL (ArangoDB Query Language) support
- Query optimization and planning
- Execution plan generation
- Cost-based optimization
- Query result caching
