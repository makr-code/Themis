# Transaction Module

SAGA pattern and transaction management implementation for ThemisDB.

## Components

- SAGA implementation
- Transaction manager
- SAGA logger
- Distributed transaction coordination

## Features

- SAGA pattern for distributed transactions
- Compensating transaction support
- Transaction logging and recovery
- Long-running transaction support
- Eventual consistency guarantees
