# Importers Module

Data import functionality for ThemisDB.

## Components

- PostgreSQL importer
- Custom import format handlers
- Import pipeline

## Features

- Import data from PostgreSQL databases
- Schema mapping and transformation
- Batch import operations
- Incremental import support
