# Themis Admin Tools Shared Library

Shared components and utilities for ThemisDB admin tools.

## Components

- **ApiClient/** - Shared API client for connecting to ThemisDB
- **Models/** - Common data models used across admin tools

## Usage

This library is referenced by all admin tools:
- Themis.AuditLogViewer
- Themis.ClassificationDashboard
- Themis.ComplianceReports
- Themis.KeyRotationDashboard
- Themis.PIIManager
- Themis.RetentionManager
- Themis.SAGAVerifier

## Building

Build as part of the admin tools solution or individually using .NET SDK.
