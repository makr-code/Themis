# Themis Key Rotation Dashboard

Desktop application for managing encryption key rotation in ThemisDB.

## Features

- View encryption key status
- Trigger key rotation
- Monitor key lifecycle
- HSM integration status

## Building

Build using .NET SDK or Visual Studio.
