# Themis Compliance Reports

Desktop application for generating compliance reports from ThemisDB.

## Features

- Generate GDPR compliance reports
- Generate HIPAA compliance reports
- Custom report templates
- Export reports to PDF/Excel

## Building

Build using .NET SDK or Visual Studio.
