# Themis Audit Log Viewer

Desktop application for viewing and analyzing ThemisDB audit logs.

## Features

- View audit log entries
- Filter by user, operation, timestamp
- Export audit reports
- Compliance verification

## Building

Build using .NET SDK or Visual Studio.
