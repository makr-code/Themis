# Themis PII Manager

Desktop application for managing Personally Identifiable Information (PII) in ThemisDB.

## Features

- Detect PII in documents
- Configure PII detection rules
- Apply pseudonymization
- Generate PII reports

## Building

Build using .NET SDK or Visual Studio.
