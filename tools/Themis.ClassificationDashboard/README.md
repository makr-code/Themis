# Themis Classification Dashboard

Desktop application for managing data classification and labeling in ThemisDB.

## Features

- View and manage data classifications
- Apply classification labels to documents
- Define custom classification schemes
- Compliance reporting

## Building

Build using .NET SDK or Visual Studio.
