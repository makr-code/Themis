# Themis SAGA Verifier

Desktop application for verifying and debugging SAGA transactions in ThemisDB.

## Features

- View SAGA transaction status
- Verify compensating transactions
- Debug failed SAGAs
- Generate SAGA reports

## Building

Build using .NET SDK or Visual Studio.
