# ThemisDB JavaScript/TypeScript SDK (Entwurf)

Dieses Paket ist derzeit nur ein Skeleton. Die eigentliche Client-Implementierung wird in zukünftigen Iterationen ergänzt.

## Skripte

| Script | Zweck |
| --- | --- |
| `npm run build` | TypeScript nach `dist/` kompilieren |
| `npm run lint` | ESLint ausführen |
| `npm run test` | Vitest-Tests starten |

> **Hinweis:** Abhängigkeiten sind in `package.json` definiert, aber noch nicht installiert. Die Implementierung läuft vor allem über HTTP (fetch) und benötigt später ein konfigurierbares Topologie-Backend.
